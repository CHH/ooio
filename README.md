# OOIO

_Object oriented IO for PHP._

## Install

### Install with [composer][]

Get Composer:

    % wget http://getcomposer.org/composer.phar

Add this to your project's `composer.json`:

    {
        "require": {
            "chh/ooio": "*"
        }
    }

Then run composer:

    % php composer.phar install
